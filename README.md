Who Faster

Примерная структура проекта как! Необходимо ознакомиться! 

Структура проекта:

- `who_faster` — основной пакет игры.
  - `config.py` — настройки игры (порт, хост, время раунда и т.д.).
  - `models.py` — базовые структуры данных (игрок, состояние игры, результаты раундов).
  - `word_list.txt` — список слов для игры.
  - `game/` — игровая логика (раунды, подсчёт очков).
  - `net/` — сетевое взаимодействие (сервер, клиент, протокол сообщений).
  - `server_app/` — точка входа для сервера.
  - `client_app/` — точка входа для клиента.
