# 🎮 Проект «Чи рӕвздӕр»
## 📄 Техническое задание

---

## 1. 🎯 Назначение разработки

**«Языковая дуэль»** — это образовательно-развлекательная игра для двух игроков, направленная на изучение **осетинского языка**.

### 🖥️ Интерфейс
Игровое поле разделено на три ключевые зоны:
| Зона | Описание |
| :--- | :--- |
| **Слева** | Таблица слов для **Игрока 1** |
| **Справа** | Таблица слов для **Игрока 2** |
| **По центру** | Изображение (задание) |

### 🏆 Цель раунда
Быстрее соперника выбрать слово в своей таблице, соответствующее картинке по центру экрана.  
Если правильного варианта в таблице нет — необходимо выбрать опцию **«Нет такого слова»**.

### 📊 Структура игры
*   **Всего раундов:** 10
*   **Максимум баллов за раунд:** 100
*   **Механика преимущества:** Игрок, ответивший первым, получает преимущество.
*   **Таймер для второго:** Если какой-то игрок ответил первым, то у второго запускается ограниченный таймер (например, `5 секунд`).
    *   ❌ **Если игрока, у которого запустился таймер, истечет время или он ответит неправилньо:** 0 баллов.
    *   ✅ **Если он ответил правильно за время таймера:** Сниженное количество баллов (расчёт по формуле).
    *   ⚠️ **Замечание:** Если игрок, ответивший первым, ошибся в своем выборе, то второй игрок, ответив правильно за время таймера, все равно получит 100 баллов, а не по формуле

---

## 2. 👥 Целевая аудитория

Игра предназначена для **людей, изучающих осетинский язык**.  
Формат игры: **Local Multiplayer** (игра с другом на одном телефоне).

---

## 3. ⚙️ Требования к функционалу

### 3.1. 🎨 Пользовательский интерфейс
*   **Чёткое визуальное разделение** зон игроков для быстрой ориентации.
*   **Информационные индикаторы:**
    *   Текущий счёт для каждого игрока.
    *   Номер текущего раунда.
    *   Таймер (при активации).

### 3.2. ⌨️ Управление с клавиатуры (На пк)

| Игрок | Навигация | Выбор |
| :--- | :--- | :--- |
| **Игрок 1** | `↑` `↓` (Стрелки) | `Enter` |
| **Игрок 2** | `W` `S` | `Пробел/Shift` |
| **Общее** | — | `Esc` (Выход в меню) |

### 3.3. 🚫 Вариант «Нет такого слова»
*   В каждой таблице обязательна ячейка с текстом **«Нет такого слова»**.
*   Выбор этой ячейки засчитывается как **правильный ответ**, если слова действительно отсутствует в таблице.

### 3.4. ⏱️ Механика таймера
*   Таймер для второго игрока запускается автоматически **после выбора первого игрока**.

### 3.5. 🧮 Формула подсчёта баллов

| Ситуация | Условия | Баллы |
| :--- | :--- | :--- |
| **Первый ответ** | Ответил первым и правильно | `100` |
| **Второй ответ** | Ответил вторым и правильно (в течение таймера) | `100 × (прошедшее_время / лимит_таймера)` |
| **Ошибка** | Неверный выбор | `0` |
| **Тайм-аут** | Не ответил за время таймера | `0` |

> **Примечание:** Количество баллов для второго игрока зависит от времени реакции.

### 3.6. 🛠️ Редактор контента
Административная панель должна позволять:
1.  **Добавить** новую запись: `Изображение` + `Осетинское название`.
2.  **Отредактировать** существующую запись (картинку или название).
3.  **Удалить** запись из базы данных.


Примерная структура проекта как! Необходимо ознакомиться! 

Структура проекта:

- `who_faster` — основной пакет игры.
  - `config.py` — настройки игры (порт, хост, время раунда и т.д.).
  - `models.py` — базовые структуры данных (игрок, состояние игры, результаты раундов).
  - `word_list.txt` — список слов для игры.
  - `game/` — игровая логика (раунды, подсчёт очков).
  - `net/` — сетевое взаимодействие (сервер, клиент, протокол сообщений).
  - `server_app/` — точка входа для сервера.
  - `client_app/` — точка входа для клиента.
